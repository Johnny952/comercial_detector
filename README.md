# Librerias
Se utilizan las siguientes librerias:
1. opencv
2. sklearn
3. numpy


# Modo de uso
Para ejercutar el script, abrir la consola de comandos y dirigirse a la carpeta contenedora del script y ejecutar `python detector.py [television.mp4] [comerciales]`, donde `television.mp4` es la direccion del archivo de video que se quiere analizar y `comerciales` la carpeta contenedora de los comerciales.

# Anexo
El script se ejecutó en un computador con windows 10 pro y con las siguientes especificaciones:
1. Intel Core i5 6400
2. 16 Gb de RAM
Y se demoró entre 8 a 10 minutos en procesar todos los comerciales mas uno de los videos.
